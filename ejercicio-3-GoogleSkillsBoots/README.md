
# Ejercicio 3 – Google Cloud Dataprep (Google Skills Boost)

Este ejercicio corresponde a un laboratorio guiado de Google Skills Boost, donde se implementa un pipeline de transformación de datos utilizando servicios de Google Cloud:

BigQuery como origen y destino de datos.

Cloud Dataprep / Alteryx Designer Cloud para exploración, limpieza y enriquecimiento.

Dataflow como motor de ejecución de los jobs generados por Dataprep.

El objetivo principal fue construir un pipeline de transformación, detectar problemas de calidad de datos, preparar métricas clave y cargar el resultado final en BigQuery.




⚙️ 1. Objetivos del trabajo
Explorar datos de sesiones de e-commerce almacenados en BigQuery.

Evaluar la calidad de los datos.

Construir un pipeline visual de limpieza y transformación con Cloud Dataprep.

Ejecutar el flujo sobre Dataflow y cargar el resultado en una tabla analítica en BigQuery.

Contestar preguntas técnicas del laboratorio.

Proponer una arquitectura alternativa basada en Google Cloud.
